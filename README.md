# Job Portal System 

A desktop-based Job Portal application built using **JavaFX**, **Spring Boot**, and **MySQL**.

This project simulates a real-world job portal where users can register, log in, and interact with job postings. Recruiters can post jobs, and job seekers can explore available opportunities.

---

##  Key Features

-  User Registration and Login system  
-  Role-based access (Job Seeker / Recruiter)  
-  Recruiters can post job openings  
-  REST APIs built using Spring Boot  
-  Data stored and managed using MySQL  
- Interactive desktop interface using JavaFX  

## Tech Stack

- **Frontend:** JavaFX  
- **Backend:** Spring Boot  
- **Database:** MySQL  
- **Build Tool:** Maven  
- **Language:** Java  

## How It Works

1. User registers with basic details  
2. User logs in with credentials  
3. Based on role:
   - Recruiter → can post jobs  
   - Job Seeker → can view opportunities  
4. Data is stored in MySQL database  
5. JavaFX GUI communicates with backend via REST APIs  

## How to Run the Project

### Step 1: Run Backend
- Open project in IntelliJ  
- Run `JobportalApplication`  
- Ensure server starts on port `8080`

### Step 2: Run Frontend (JavaFX)
- Run `JobPortalGUI.java`  

## Database Configuration

- Create a MySQL database 
  
- Update your database credentials
  
- Ensure MySQL server is running before starting backend
